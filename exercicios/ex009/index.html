Listas:

Listas Ordenadas:

Tags: <ol> e <li>

Uma ol tem importância na ordem de acontecimento das coisas, como uma receita de bolo.
Por exemplo:

  1. Acordar
  2. Ir a escola
  3. Voltar para casa
  4. Dormir

Listas não Ordenadas:

Tags:<ul> e <li>

uma ul não tem ordem, como uma lista de mercado.
Por exemplo:

  • feijao
  • arroz
  • massa
  • carne

Obs: ambas as listas tem parametros types para mudar a forma ou o objeto de numerar.

Junção de listas

Também podemos juntar listas.
por exemplo:

  1. Acordar
  2. Ir a escola
  3. ir ao mercado e comprar:
    • feijao
    • arroz
    • massa
    • carne
  4. Voltar para casa
  5. Dormir

lista de Definições

Tags: <dl> e <dt> e <dd>

Usada para defenir termos.
Exemplo:

HTML
Linguagem de marcação para criação de conteúdos em sites.
CSS
Linguagem de marcação para o desing de um site.
JavaScript
Linguagem de programação para a criação de interações em um site.
