Links Externos

Um link externo leva para outro site na mesma janela.
Exemplo:

O canal Ciência Todo Dia é como o nome diz um canal de ciencia.
O jeito acima foi usando apesnas um link sem nenhum atributo, quando fazemos assim o novo site é aberto na mesma aba do seu, que acaba o fechando, usando o atributo target_blank abrimos em outra janela o link.
Veja:

O canal Ciência Todo Dia é como o nome diz um canal de ciencia.
É recomendado fazer isso quando temos links externos.
