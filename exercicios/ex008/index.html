Principais tags de formatações:

Negrito ou destaque

Para transformarmos uma frase em negrito devemos usar a tag: <strong>
Por exemplo: esse trecho é um destaque por isso esta em negrito!

Ou também podemos usar uma tag não semântica <b>
Isso quando não queremos destaque, apenas forma.

Ítalico ou ênfase

Para fazermos uma frase em italico usamos a tag: <em>
Por exemplo: esta em italico pois tem enfase nessa frase!

Quando queremos apenas forma usamos a tag <i>

Marca texto

Para marcarmos uma frase usamos a tag: <mark>
para mudarmos a cor do marcador usamos a tag style, se fizermos isso no cabeçalho vale para todas as tags selecionadas.

Exemplo: frase marcada para destaque!

Tamanho de texto

Para mudar o tamango do texto temos a tag: <big> e <small>

A tag big esta obsoleta e não deve ser usada ja a small pode.

Texto excluido e incluido

para fazer um texto excluido usamos a tag:<del>

para fazer um texto incluido usamos a tag:<ins>

O del deve ser lido mas nao considerado abobora
O ins deve ter enfase!

Potencia e demencia

As tags sao: <sup> e <sub>
